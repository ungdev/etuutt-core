    Back 🧠 💾

    PHP est un language tourné vers le traitement des requêtes web. Il en existe d'autres qui peuvent remplir cette fonction (NodeJS, Python...).

    Nous allons utiliser Symfony pour sa simplicité et sa communauté. Nous ajoutons la librairie Symfony API Platform pour avoir une API fonctionnelle qu'il nous reste à configurer.

    PHP est très puissant, mais encore plus avec Symfony, un ensemble de fonction founissant un cadre de travail (framework) très pratique pour les développeurs. * Introduction à la gestion des packages et à Composer dans le dossier Composer * Introduction aux API * Symfony : doc dans le dossier Symfony * API Platform : Deux liens pour apprendre, SymfonyCasts et Grafikart (Cette dernière est plus complète, mais il faut déjà avoir suivi des tutos Symfony avant)
